# ZS Minimap – Smart Radar for FiveM

ZS Minimap es un script standalone que controla de forma inteligente la aparición del minimapa (radar) en FiveM.

## ✨ Características
- Minimap oculto al caminar
- Minimap visible solo en vehículos
- Compatible con autos, motos, bicicletas y patinetas
- Sin frameworks (Standalone)
- Ultra optimizado (bajo consumo)
- No usa NUI
